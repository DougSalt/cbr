cbr:get-rank

Inputs

Outputs

This reports an integer as a NetLogo number indicating the rank of the case within the case-base. This is by default zero unless otherwise set by cbr:set-rank. This is used to detemine case precedence if there is a draw when trying to match a case-base.

Purpose

Example

cbr:set-rank

Inputs

  • A case base of the type cbr:new

  • An integer rank of 0 or above.

Purpose

This is a command and has no return. It sets the rank of a particular case in a case-base. If, when matching cases, two cases are indentical, then it is the case with the highest rank that will be selected. If two or more cases are matched and have the same rank, then if the match is made using cbr:match, then a random selection from the matching cases is returned. If cbr:matches is used, then all the matches with the same rank are returned.

The default rank is always 0, and the value may be interrogated using cbr:get-rank.
